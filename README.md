# PinSpace

A Pinterest-inspired full stack web application built using React, Express, PostgreSQL, and Prisma.

## Features

* User Authentication (JWT)
* Responsive Masonry Feed
* Create and View Posts
* User Profile Dashboard
* Like Interactions
* Responsive UI

## Tech Stack

### Frontend

* React
* React Router
* Tailwind CSS
* Axios
* Framer Motion

### Backend

* Node.js
* Express.js
* PostgreSQL
* Prisma ORM
* JWT Authentication

## Getting Started

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```
